浏览器的history对象

Posted on 2019-03-22 | In web

history对象的作用

用来管理当前窗口的浏览历史,只能看浏览过多少页面不能看具体内容.

常用方法
  • back 页面回退
  • foward 前进一个页面
  • go(数字) 正的为前进.负数为倒退
  • pushState 给history增加一个状态,浏览器地址会发生变化,但是不会发生跳转
    • replaceState 用来修改当前history对象.
事件

popState 当浏览器url发生变化时会触发这个事件 回调函数上参数event上的state 是pushState 和 repalaceState添加的
页面第一次加载不会触发popSate事件
